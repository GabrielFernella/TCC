# Anotações

## Pacotes Iniciais

1. yarn add jest -D (Ferramenta para elaboração de testes)
2. yarn jest --init (Para criar o arquivo jest)
3. yarn add ts-jest -D (extensão para o jest entender typescript)
4. yarn add @types/jest -D (Faz com que nosso projeto entenda as tipagens destinadas a testes)
5. yarn test (Executa todos os testes)
